# Вторая порядковая статистика (Sparse Table)

## Описание задачи
В этой задаче рекомендуется использовать Sparse Table. Время ответа на запрос должно составлять O(1).

Дан массив из `n` чисел. Обработайте `q` запросов следующего вида:
* по индексам `l` и `r` найдите значение второй порядковой статистики на отрезке массива `[l, r]`.

## Формат ввода
Первая строка содержит два целых числа `n` и `q` (`2 <= n <= 10^5`, `1 <= q <= 10^5`).

Вторая строка содержит `n` целых чисел `a_1, a_2, ..., a_n` (`|a_i| <= 10^9`).

Каждая из следующих `q` строк содержит описание очередного запроса: два целых числа `l` и `r` (`1 <= l < r <= n`), задающих границы отрезка.
