# D. Быстрое преобразование Фурье

## Описание задачи
Даны два многочлена $A(x) = a_nx^n + a_{n-1}x^{n-1} + \dots + a_0$ и $B(x) = b_mx^m + b_{m-1}x^{m-1} + \dots + b_0$.
Вычислите их произведение $C(x) = A(x) \cdot B(x)$.

## Формат ввода
Первая строка входных данных содержит описание многочлена $A$ в следующем виде: сначала целое число $n$ ($0 \le n < 2^{16}$), а затем $(n + 1)$ целое число $a_n, a_{n-1}, \dots, a_0$ ($-9 \le a_i \le 9, a_n \ne 0$).

Вторая строка входных данных содержит описание многочлена $B$ в аналогичном формате: сначала целое число $m$ ($0 \le m < 2^{16}$), а затем $(m + 1)$ целое число $b_m, b_{m-1}, \dots, b_0$ ($-9 \le b_i \le 9, b_m \ne 0$).

## Формат вывода
Выведите описание многочлена $C(x)$ в таком же формате: сначала степень многочлена, а затем его коэффициенты от старшего к младшему через пробел.
