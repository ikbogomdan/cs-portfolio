## D. LCS

### Описание задачи
Даны две строки. Найдите их наибольшую общую подпоследовательность.

### Формат ввода
Первая строка содержит строку $s$ ($1 \le |s| \le 10^3$). 
Вторая строка содержит строку $t$ ($1 \le |t| \le 10^3$).
Обе строки состоят из маленьких латинских букв.

### Формат вывода
В первой строке выведите $k$ — длину наибольшей общей подпоследовательности строк $s$ и $t$.
Во второй строке выведите $k$ индексов строки $s$, задающих эту подпоследовательность.
В третьей строке выведите $k$ индексов строки $t$, задающих эту подпоследовательность.
Если существует несколько решений, выведите любое из них.

## F. Цветной рюкзак

### Описание задачи
Даны $n$ предметов. Каждый из них имеет свой вес $w_i$, стоимость $c_i$, а также цвет $f_i$.

Назовём набор предметов пёстрым, если все предметы в нём имеют попарно различные цвета. 

Также есть рюкзак прочности $m$. Вы можете сложить некоторые из предметов в рюкзак, но только при условии, что суммарный вес выбранных предметов не превосходит $m$, и при этом набор должен являться пёстрым. Какую максимальную стоимость может иметь такой набор?

### Формат ввода
Первая строка содержит три целых числа $n$, $g$ и $m$ ($1 \le g \le n \le 300$, $1 \le m \le 10^4$) — число предметов, количество различных цветов, а также вместимость рюкзака.

В $i$-й из следующих $n$ строк содержится три целых числа $w_i$, $c_i$ и $f_i$ ($1 \le c_i \le w_i \le 10^4$, $1 \le f_i \le g$).

## K. Невозрастающая подпоследовательность

### Описание задачи
Найдите наибольшую невозрастающую подпоследовательность в данном массиве.

Формально, дан массив $a_1, a_2, \dots, a_n$. Найдите последовательность ($i_1, i_2, \dots, i_k$) максимальной длины, такую что $i_1 < i_2 < \dots < i_k$ и при этом $a_{i_1} \ge a_{i_2} \ge \dots \ge a_{i_k}$.

### Формат ввода
Первая строка содержит одно целое число $n$ ($1 \le n \le 250\,000$).
Вторая строка содержит $n$ целых чисел $a_1, a_2, \dots, a_n$ ($|a_i| \le 2^{31} - 2$).

### Формат вывода
Сначала выведите длину найденной подпоследовательности, а затем список её индексов.

## P. Путешественник на Марс

### Описание задачи
Поверхность Марса можно представить в виде плоскости с декартовой системой координат. Движение по Марсу очень ограничено: из точки $(x, y)$ переместиться можно только в точки $(x, y - 1)$ и $(x + 1, y - 1)$. 

На Марсе расположено $n$ лётных площадок. До $i$-й из них с Земли можно добраться любым из $k_i$ космических рейсов. 

Также на Марсе есть $q$ волшебных прямых, задаваемых уравнениями $y = x + c_j$.

Маршрут путешественника описывается следующим образом:
1. Выбирается одна из $n$ лётных площадок.
2. Путешественник прилетает на неё с помощью одного из доступных космических рейсов.
3. Затем, двигаясь по поверхности Марса, доходит до произвольной точки на прямой.

Для каждой прямой путешественник хочет определить число различных способов, которым можно с Земли добраться до этой прямой. Поскольку ответ может быть очень большим, выведите его по модулю $10^9 + 9$.

## U. Клики

### Описание задачи
Неориентированный граф на $n$ вершинах задан своей матрицей смежности. Найдите количество клик в нём (включая пустое множество).

### Формат ввода
Первая строка содержит одно целое число $n$ ($1 \le n \le 50$). 

Следующие $n$ строк содержат матрицу смежности графа: в $i$-й строке $j$-й символ равен «1», если вершины $i$ и $j$ соединены ребром, иначе символ равен «0». Гарантируется, что матрица симметрична и на её главной диагонали стоят нули.

### Формат вывода
Выведите количество клик в заданном графе.
