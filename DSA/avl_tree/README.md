# Lower Bound (AVL)

**В данной задаче вам нужно реализовать AVL-дерево. Другие решения не принимаются.**

Пусть S — множество чисел. Изначально S пусто. Обработайте запросы двух видов:
* (запрос «+») `insert x`: добавить число x в S. Если x уже есть в S до запроса, то запрос игнорируется.
* (запрос «?») `lower_bound x`: найти в множестве S наименьшее число, не меньшее x. Если такого числа нет, выведите -1.

На запросы нужно отвечать онлайн (то есть следующий запрос становится известен только после ответа на предыдущий).

## Формат ввода
Первая строка содержит одно целое число q (1 <= q <= 3 * 10^5) — количество запросов. Следующие q строк содержат описание запросов.

Запрос вида «? x» задаёт операцию `lower_bound x`.

Если запрос вида «+ x» является первым запросом или идёт во входном файле после другого запроса вида «+», то он задаёт операцию `insert x`. Если же он идёт после запроса вида «?», ответ на который был равен y, то выполняется операция `insert (x + y) % 10^9`.

Во всех запросах x является целым числом в диапазоне [1, 10^9].
