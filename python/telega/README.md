Цель проекта - упростить сбор информации с различных сайтов посредством selenium. В качестве "api" выступает телеграмм бот, в которой есть
необходимая инструкция по работе с ботом, а так же некоторые опции/фильтры, выбор которых влияет на результат по парсингу того или иного сайта. 
Главная цель проекта -  упростить сборку, которая является исключительно механической работой, где всегда надо сделать 2 вещи: 
1) Сохранить все фотографии той или иной позиции
2) Оформить это с некоторым шаблоном (он меняется в зависимости от условий доставки по той или иной стране или же, например, от оформления по сайту), 
отправив все это в телеграмм 

Модули, которые используются в проекте: 

Для телеграмм бота использовалась библиотека aiogram в сочетании с asyncio

Для класса, которая реализовывал общую логику по парсингу сайтов использовались: 
undetected_chromedriver 
selenium
requests 
PIL, io - для работы с изображениями 

Для класса, который реализовывал запуск программ: 
import shutil
import sys
