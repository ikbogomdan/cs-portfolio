# EARLEY Algorithm

Реализация алгоритма Earley.

## Файлы
* `main.cpp` - Основная программа для ручного ввода или тестирования системой.
* `run_tests.py` - Программа на питоне, которая запускает тесты и делает вывод, в зависимости от вердикта

## Сборка и запуск


### Основная программа
```bash
g++ -std=c++17 main.cpp -o main
./main
```
### Тесты
```bash
make test
```
