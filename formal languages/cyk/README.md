# CYK Algorithm

Реализация алгоритма Кока — Янгера — Касами с приведением к НФХ 

## Файлы
* `cyk.hpp` - Заголовочный файл с классом алгоритма.
* `main.cpp` - Основная программа для ручного ввода или тестирования системой.
* `tests.cpp` - Unit-тесты с жестко заданными грамматиками.

## Сборка и запуск

### Основная программа
g++ -std=c++17 main.cpp -o main
./main

### Тесты
g++ -std=c++17 tests_run.cpp -o run_tests
./run_tests
