# Gitfame: Git Repository Analytics Tool

**Gitfame** — это консольная утилита на Go для глубокого анализа авторства 
и подсчета статистики контрибьюторов в Git-репозиториях. 

Инструмент парсит Git Object Database и рассчитывает вклад каждого 
разработчика на основе актуального состояния файлов. Обработка файлов 
распараллелена, что обеспечивает высокую 
скорость анализа даже на объемных монорепозиториях.

## Ключевые возможности

* Подсчет количества строк, коммитов и файлов для каждого автора.
* Анализ состояния репозитория на момент любого коммита (revision).
* Параллельная обработка команд `git blame` через горутины.
* Гибкая фильтрация файлов: по расширениям, языкам и Glob-паттернам.
* Поддержка форматов вывода: JSON, CSV, JSON-Lines и Tabular.

## Установка

Для сборки и установки бинарного файла выполните команду:

`go install ./cmd/gitfame/...`

## Использование

Базовый запуск в текущей директории:

`gitfame --repository=. --extensions='.go,.md' --order-by=lines`

### Поддерживаемые флаги

* `--repository` : путь до Git репозитория (по умолчанию `.`).
* `--revision` : указатель на коммит или тег (по умолчанию `HEAD`).
* `--order-by` : ключ сортировки результатов (`lines`, `commits`, `files`).
* `--use-committer` : считает коммиттеров вместо изначальных авторов.
* `--format` : формат вывода (`tabular`, `csv`, `json`, `json-lines`).
* `--extensions` : фильтр по расширениям (например, `.go,.md`).
* `--languages` : фильтр по языкам программирования (`go,markdown`).
* `--exclude` : Glob-паттерны для исключения файлов (`vendor/*`).
* `--restrict-to` : Glob-паттерны для строгого включения файлов.

## Архитектура и стек

* **Язык:** Go (Golang)
* **Конкурентность:** Goroutines, Channels (Worker Pool для балансировки).
* **Интеграция:** `os/exec` для безопасного вызова подпроцессов Git.
