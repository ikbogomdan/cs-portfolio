# cs-portfolio
Computer Science. Алгоритмы и структуры данных (Python, C++) плюс низкоуровневое системное программирование и архитектура (C, Assembly).

## Навигация по проектам

* **[C++](./C++)**: объектно-ориентированное программирование, ручное управление памятью и реализация STL-совместимых контейнеров.
* **[C](./C)**: системное программирование и решение низкоуровневых задач.
* **[DSA](./DSA)**: реализация классических алгоритмов и сложных структур данных.
* **[Formal Languages](./formal%20languages)**: реализация алгоритмов Кока-Янгера-Касами (CYK) и Эрли для синтаксического анализа.
* **[Assembler](./assembler)**: низкоуровневая разработка и работа с архитектурой процессора.
* **[Python / Telega](./python/telega)**: асинхронный Telegram-бот для автоматизированного парсинга данных.
